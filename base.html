基础css

crm的样式部分目前分为基础样式（base）和常用组件（widget），这里首先介绍基础样式。

本部分主要定义基础样式：包括（1）全局sass定义 (2) 重置样式，（3）全局样式，以及（4）基础的布局样式。

注意：开发组件或页面时必须引用base

全局sass定义由于css开发使用了sass方式，所以在base中首先定义了sass的全局变量和mixin，供组件和页面开发使用

重置样式屏蔽一些浏览器的差异性，参考pc的reset，根据无线浏览器做适当修改。
